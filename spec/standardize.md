# 코드 표준화 규칙

키 필드(강사코드, 강의실코드, 교과목코드)는 아래 규칙으로 표준화합니다.

1. `trim` 적용
2. 연속 공백 축소
3. 제어문자 제거
4. 구분자(괄호, 언더스코어, 대시, 슬래시, 기호 등) 일관화
   - 최종 구분자는 단일 `_`로 통일
5. `이름_숫자코드` 패턴 유지
   - 이름/코드 사이 공백 제거

예시:

- `"  홍길동 - 1001  "` -> `"홍길동_1001"`
- `"ROOM ( 01 )"` -> `"ROOM_01"`
- `"SUBJ/AI-01"` -> `"SUBJ_AI_01"`
